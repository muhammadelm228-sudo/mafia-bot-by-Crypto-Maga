# 🎮 Mafia Telegram Bot

Telegram-бот для игры в Мафию, написанный на **aiogram 3.x**

---

## 🚀 Возможности

- 🎭 Автоматическая раздача ролей
- 🔪 Мафия
- 💉 Доктор
- 🕵 Комиссар
- 🌙 Ночная фаза
- ☀️ Дневное голосование
- 🏆 Автоматическая проверка победы
- 👥 Поддержка нескольких чатов одновременно

---

## 🛠 Используемые технологии

- Python 3.10+
- aiogram 3.x
- Asyncio

---

## 📦 Установка

### 1️⃣ Клонировать репозиторий

```bash
git clone https://github.com/muhammadelm228-sudo/mafia-telegram-bot.git
cd mafia-telegram-bot
```

### 2️⃣ Создать виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

### 3️⃣ Установить зависимости

```bash
pip install -r requirements.txt
```

### 4️⃣ Создать .env файл
